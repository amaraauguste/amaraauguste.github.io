a) Write a Java program that prompts the user to enter the number whose factorial they wish to compute and then calculates the factorial of that number using a while loop.

The factorial of any number n:

  • is represented by n!
  • and is equal to 1*2*3*....*(n-1)*n
Example: the factorial of 5 is represented as 5!

And, 5! = 5 * 4 * 3 * 2 * 1 = 120

Note: The value of 0! is 1, according to the convention for an empty product.

Here is a step-by-step walkthrough to help you outline a solution:

  • First, prompt the user for the number that they want to reverse

  • Then, create the variables:
    • reverseNum, initialized to 0, to hold the reversed number
    • rem, initialized to 0, to temporarily hold digits as we extract them

  • Next, use a while loop to extract each digit from the number and build the reversed number:
    • Extract the last digit of num by using modulus operator (%) with 10 and store it in rem
    • Build the reversed number by multiplying reverseNum by 10 and adding rem to it
    • Remove the last digit from num by performing integer division (/) by 10

  • Finally, print out the reversed number

b) Show what is printed by the following program:

The data values to be used are as follows: 3 6 9 12 0 15 2 5 8 11

public static void main(String [] args){
        String x;
        int month;
        Scanner sc = new Scanner (System.in);
        
        do {
            System.out.print("Enter any number: ");
            month = sc.nextInt();
            x = season(month);
            System.out.println(month + " " + x);
        } while (month != 8);
    
    }
    
    public static String season(int month){
        String [] choices = {"winter", "spring", "summer", "fall", "error"};
    
        if (month == 1 || month == 2 || month ==  12) {
            return choices[0];
        } else if (month >= 3 && month <= 5) {
            return choices[1];
        } else if (month >= 6 && month <= 8) {
            return choices[2];
        } else if (month >= 9 && month <= 11) {
            return choices[3];
        } else {
            return choices[4];
        }
    }
